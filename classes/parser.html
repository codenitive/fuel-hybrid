    Parser

    The Parser class acts as an object wrapper for HTML formatted string written using format such as Markdown, Textile, BBCode or etc.

    Available Drivers:

    • Markdown
    • Textile
    • BBCode

    forge($name = null)

    Creates a new instance of the either Markdown, Textile or BBCode parser, or load from cache if already loaded.

    Static Yes
    Parameters
    Param Default Description
    $name null null to fetch the default driver, or a driver id to get a specific one
    Returns string
    Example 
    // Initiate markdown parser
$markdown = \Hybrid\Parser::forge('markdown');

echo $markdown->parse('### Hello world'); // return '<h3>Hello World</h3>'

\Hybrid\Parser::forge('BBCode');

    parse($text = '')

    Parse given formatted text to HTML.

    Static No
    Parameters
    Param Default Description
    $text '' A string for formatted text to be parse.
    Returns string
    Example 
    \Hybrid\Parser::forge('markdown')
    ->parse('Some ```other code```'); // return 'Some <code>Hello World</code>'
