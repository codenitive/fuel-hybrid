    Auth OAuth

    The Auth classes provides a standardized interface for authentication in Fuel. This allows our users to write their own drivers and easily integrate a new driver to work with old code by keeping the basic methods consistent.

    How To

    1. Install OAuth and OAuth2 package for FuelPHP 
      $ php oil package install oauth --direct
      $ php oil package install oauth2 --direct
    2. Both need to be included in autoload packages under APPPATH/config/config.php 
      'always_load'   => array(

    /**
     * These packages are loaded on Fuel's startup.  You can specify them in
     * the following manner:
     *
     * array('auth'); // This will assume the packages are in PKGPATH
     *
     * // Use this format to specify the path to the package explicitly
     * array(
     *     array('auth' => PKGPATH.'auth/')
     * );
     */
    'packages'  => array(
        'orm',
        'oauth',
        'oauth2',
        'hybrid',
    ),
    3. Create a controller at APPPATH/classes/controller/connect.php 
      <?php

class Controller_Connect extends \Hybrid\Auth_Controller {}
    4. Update APPPATH/config/autho.php, in `provider` section.
    5. Create a login button and you're ready to go. 
      <?php echo \Hybrid\Html::anchor('connect/session/twitter', 'Sign-in with Twitter'); ?>
