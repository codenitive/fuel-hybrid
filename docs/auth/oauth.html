Auth

The Auth classes provides a standardized interface for authentication in Fuel. This allows our users to write their own drivers and easily integrate a new driver to work with old code by keeping the basic methods consistent.

How To

  1. Create a controller at APPPATH/classes/controller/connect.php 
    <?php

class Controller_Connect extends \Hybrid\Auth_Controller {}
  2. Update APPPATH/config/autho.php, in `provider` section.
  3. Create a login button and you're ready to go. 
    <?php echo \Hybrid\Html::anchor('connect/session/twitter', 'Sign-in with Twitter'); ?>
