    Curl Class

    Curl class processes external URI requests.

    Scheduled to be replaced with \Fuel\Core\Request_Curl in Fuel v1.1

    How To

    $google = \Hybrid\Curl::forge("GET http://google.com");
$google->setopt(array(
    CURLOPT_RETURNTRANSFER => 1,
));

\Debug::dump($google->execute());

    forge($uri, $dataset = array())

    Initiate this class as a new object.

    Static Yes
    Parameters
    Param Default Description
    $url Required The URI being requested (including HTTP method). If this parameter is not present, the URI class will be used to detect the URI.
    $dataset array() An array of GET, POST, PUT or DELETE dataset (depends on which HTTP method is called)
    Returns \Hybrid\Curl Object
    Example 
    // GET method support query string
$curl = \Hybrid\Curl::forge('GET http://google.com?q=hello world');

// Alternatively
$curl = \Hybrid\Curl::forge('GET http://google.com', array('q' => 'hello world'));

    setopt($option, $value = null)

    Set curl options.

    Static Yes
    Parameters
    Param Default Description
    $option Required Either an array (key as CURL option) or a string of CURL option name
    $value '' Optional CURL option value (when
    Returns \Hybrid\Curl Object
    Example 
    $curl->setopt(CURLOPT_HEADER, true);

// or use array to assign multiple value
$curl->setopt(array(
    CURLOPT_HEADER => true,
    CURLOPT_NOBODY => false,
));

    execute()

    Execute the Curl request and return the output.

    Static No
    Parameters No
    Returns Object
    Example
