Curl Class

Curl class processes external URI requests.

To be replaced with \Fuel\Core\Rest_Curl in Fuel v1.1

How To

$google = \Hybrid\Curl::factory("GET http://google.com");
$google->setopt(array(
    CURLOPT_RETURNTRANSFER => 1,
));

\Debug::dump($google->execute());

factory($uri, $dataset = array())

Initiate this class as a new object.

Static Yes
Parameters
Param Default Description
$url Required The URI being requested (including HTTP method). If this parameter is not present, the URI class will be used to detect the URI.
$dataset 
array()
 An array of GET, POST, PUT or DELETE dataset (depends on which HTTP method is called)
Returns \Hybrid\Curl Object
Example 
// GET method support query string
$curl = \Hybrid\Curl::factory('GET http://google.com?q=hello world');

// Alternatively
$curl = \Hybrid\Curl::factory('GET http://google.com', array('q' => 'hello world'));

setopt($option, $value = null)

Set curl options.

Static Yes
Parameters
Param Default Description
$option Required Either an array (key as CURL option) or a string of CURL option name
$value 
''
 Optional CURL option value (when
Returns \Hybrid\Curl Object
Example

// or use array to assign multiple value
