    Html Class

    The Html class is an HTML wrapper for nearly all HTML tags.

    Extends

    Html class extends Fuel\Core\Html.

    title($content = '', $attributes = array())

    Generates a html title tag complete with site name from config/app.php.

    Static Yes
    Parameters
    Param Default Description
    $content '' A string of page title
    $attributes array() An array of <title> attributes
    Returns string
    Example 
    \Config::load('app', 'app');
\Config::set('app.site_name', 'FuelPHP');

$title = \Hybrid\Html::title('Hello');
// return '<title>Hello — FuelPHP</title>'

$title = \Hybrid\Html::title('');
// return '<title>FuelPHP</title>'
