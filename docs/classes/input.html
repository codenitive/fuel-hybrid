Input Class

The input class allows you to access HTTP parameters, load server variables and user agent details.

Extends

Input class extends \Fuel\Core\Input.

Additional methods in \Hybrid\Input allow to Fuel to process "HMVC + RESTful Request" using \Hybrid\Request.
