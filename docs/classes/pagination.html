Pagination Class

The pagination class allows you to easily setup pagination for records you display.

Extends

Pagination class extends \Fuel\Core\Pagination.

build_get_query($values)

Build query string, a feature missing in many pagination class. This allow you to combine search/filtering option using $_GET for your application with pagination.

Static Yes
Parameters
Param Default Description
$values Required A string or an array of $_GET attributes to be included in the pagination link
Returns String
