    Restserver Class

    Rest server class for Fuel, handle JSON, XML, CVS and etc request for endless set of Controllers instead of just Fuel\Core\Controller_Rest in Fuel Framework.

    Normally this will only be called from within \Hybrid\Controller_Rest and \Hybrid\Controller_Hybrid.

    How To

    $restserver = \Hybrid\Restserver::forge(array('text' => 'Hello World'), 200);
$restserver->format('json');
$restserver->execute();

    forge($data = array(), $http_code = 200)

    A shortcode to initiate this class as a new object.

    Static Yes
    Parameters
    Param Default Description
    $data array() The URI being requested (without HTTP method). If this parameter is not present, the URI class will be used to detect the URI.
    $http_code 200 An array of GET dataset
    Returns bool
    Example 
    $restserver = \Hybrid\Restserver::forge(array('text' => 'Hello World'), 200);

    is_rest_call()

    Check whether current request is REST.

    Static Yes
    Parameters No
    Returns bool
    Example 
    \Hybrid\Restserver::is_rest_call();

    content_type($format)

    Get content-type.

    Static Yes
    Parameters No
    Returns string
    Example 
    \Hybrid\Restserver::content_type('json'); // Return: application/json

    auth()

    Run authentication.

    Static Yes
    Parameters No
    Returns void
    Example 
    \Hybrid\Restserver::auth();

    format($rest_format = '')

    Set the REST format.

    Static No
    Parameters
    Param Default Description
    $rest_format '' A string of REST format.
    Returns void
    Example 
    $restserver->format('json');

    execute()

    Execute the REST request and return the output.

    Static No
    Parameters No
    Returns Object
    Example 
    $restserver->execute();
