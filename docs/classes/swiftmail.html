Swiftmail Class

Missing E-mail Class for Fuel Framework using Swiftmailer E-mail Library.

Information

Swiftmail can be configured in a config/email.php file and takes the following keys:

How To

\Config::load('email');

$mail = \Hybrid\Swiftmail::forge();
$mail->to('to@domain.com', 'Name');
$mail->from('Sample User', 'from@domain.com'));
$mail->subject('E-mail Title');
$mail->message('E-mail message');

if (false === $mail->send())
{
    throw new \Fuel_Exception('Unable to send email');
}

forge($config = array())

Static Yes
Parameters
Param Default Description
$config 
array()
