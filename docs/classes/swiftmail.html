    Swiftmail Class

    Missing E-mail Class for Fuel Framework using Swiftmailer E-mail Library.

    Swiftmail can be configured in a config/swiftmail.php

    How To

    \Config::load('swiftmail');

$mail = \Hybrid\Swiftmail::forge();
$mail->to('to@domain.com', 'Name');
$mail->from('Sample User', 'from@domain.com'));
$mail->subject('E-mail Title');
$mail->message('E-mail message');

if (false === $mail->send())
{
    throw new \Fuel_Exception('Unable to send email');
}

    forge($config = array())

    Creates a new instance of the Swiftmail.

    Static Yes
    Parameters
    Param Default Description
    $config array() An array to overwrite default config from config/email.php.
    Returns \Hybrid\Swiftmail Object
    Example 
    $email = \Hybrid\Swiftmail::forge();

    __construct($config = array())

    Construct a new e-mail, alias to \Hybrid\Swiftmail::forge().

    Static No
    Parameters
    Param Default Description
    $config array() An array to overwrite default config from config/email.php.
    Returns \Hybrid\Swiftmail Object
    Example 
    $email = new \Hybrid\Swiftmail;

    subject($subject)

    Sets the subject of the email.

    Static No
    Parameters
    Param Default Description
    $subject Required The emails subject.
    Returns \Hybrid\Swiftmail Object
    Example

    message($content)

    Sets the message of the email, content type is determined by 'mailtype' config.

    Static No
    Parameters
    Param Default Description
    $content Required The emails.
    Returns \Hybrid\Swiftmail Object
    Example 
    $email->message('Some message');
