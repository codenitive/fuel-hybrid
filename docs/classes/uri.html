    Uri Class

    The Uri class allows you to interact with the URI.

    Extends

    Uri class extends Fuel\Core\Uri.

    build_get_query($values, $start_with = '?')

    Build query string allow you to combine search/filtering option using $_GET for your application with \Hybrid\Pagination class.

    Static Yes
    Parameters
    Param Default Description
    $values Required A string or an array of $_GET attributes
    $start_with '?' Value of the first string, normally either ? or &
    Returns String
    Example 
    <?php
$_GET = array(
    'hello' => 'world',
    'foo'   => 'bar',
    'fuel'  => 'php',
);

echo \Hybrid\Uri::build_get_query(array('foo', 'hello')); // return '?foo=bar&hello=world'
