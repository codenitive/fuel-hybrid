View Class

The View class acts as an object wrapper for HTML pages with embedded PHP, called "views". Variables can be assigned with the view object and referenced locally within the view.

Extends

View class extends \Fuel\Core\View.

add_path($path)

Set the global path, overwrite default using /app/views/ folder.

Static Yes
Parameters
Param Default Description
$path Required A string of valid absoluted path to view
Returns void
Example 
$view = \Hybrid\View::factory('index');

// set folder location
$view->set_path(DOCROOT . 'views/');

// render the view
$this->response->body = $view->render();    // this will render DOCROOT/views/index.php
