Parser Class

The View class acts as an object wrapper for HTML formatted string written using format such as Markdown, Textile, BBCode or etc.

forge($name = null)

Static Yes
Parameters
Param Default Description
$name 
null
 null to fetch the default driver, or a driver id to get a specific one
Returns string
Example 
// Initiate markdown parser
$markdown = \Hybrid\Parser::forge('markdown');

echo $markdown->parse('### Hello world'); // return '<h3>Hello World</h3>'

echo $markdown->parse('Some ```other code```'); // return 'Some <code>Hello World</code>'

Available Drivers:

  • Parser_Markdown
  • Parser_Textile
  • Parser_BBCode
