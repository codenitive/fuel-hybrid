    Installation Instructions

    Manual Installation

    This will create the default installation of the Fuel Hybrid in your Fuel Framework installation.

    1. Download the Hybrid for FuelPHP
    2. Unzip/Extract the download
    3. Move the files to your server, extracted files should be added to fuel/packages/hybrid 
      /
  fuel/
	app/
	core/
	packages/
	  hybrid/
		classes/
		config/
		docs/
		lang/
		tasks/
		tests/
		vendor/
		bootstrap.php
		README.md
  public/
	.htaccess
	assets/
	index.php
  oil
    4. Fuel Hybrid and Orm package need to be included in the autoload packages under APPPATH/config/config.php 
      'always_load'   => array(

	/**
	 * These packages are loaded on Fuel's startup.  You can specify them in
	 * the following manner:
	 *
	 * array('auth'); // This will assume the packages are in PKGPATH
	 *
	 * // Use this format to specify the path to the package explicitly
	 * array(
	 *     array('auth' => PKGPATH.'auth/')
	 * );
	 */
	'packages'  => array(
		'orm',
		'hybrid',
	),
